## 二、两类业务说明

### 2.1 鲜奶（教职工订阅）

- **产品**：150ml 燕塘巴氏鲜奶 / 燕塘乳酸奶
- **订阅单位**：1 份 = 100 支
  - 仅订鲜奶：100 支鲜奶
  - 仅订乳酸：100 支乳酸
  - 鲜奶+乳酸均订：50 支鲜奶 + 50 支乳酸
- **教职工优惠价**：2.1 元/支（市场原价 3.5 元，6 折优惠）
- **配送节奏**：每周 2 次
  - 周日备货 → **周一配送 3 天量**（每份 3 支）
  - 周三备货 → **周四配送 4 天量**（每份 4 支）

### 2.2 常温奶（家长订阅学生奶）

- **产品**：多口味学生奶（纯奶 / 乳酸 / 甜奶 / 草莓 / 麦香），125ml 或 200ml
- **订阅单位**：1 套 = 100 支（家长线上下单）
- **配送方式**：统一送至学校奶室，由班主任或奶室负责人按班分发
- **配送频次建议**：周供量 ≥ 50 支的学校每周配送；量少学校两周合并一次

## 五、操作注意事项

1. **配送量以订单文件为准**：第③④表中标注"订单导入"的数据来源于 `3天/4天鲜奶量录入明细总表.xlsx`，此为 ERP 导入格式，最接近实际执行量；"计算估算"是根据征订份数推算的近似值，建议与各学校确认后使用。

2. **退订/新增处理**：学期中出现退订或新增征订时，需更新对应学校的源文件后重新运行 `build_report.py` 脚本，总表会自动更新。

3. **常温奶口味备货**：常温奶口味比例因各校套餐组合不同而有差异。第④表中各口味周量仅为参考估算，建议结合第①列"纯牛奶份数/多口味份数"进行精确备货。

4. **补货计划日历**：第②表中的逐周日历，每次配送前请核对"黄色本周"行的备货数量是否与最新订单匹配，如有调整直接修改源订单文件后重跑脚本。

5. **数据更新频率**：
   - 鲜奶：建议每次新学期开学、或每月初核实一次征订名单
   - 常温奶：以学校系统导出的配货表为准，每次出货前更新
